Install from the browser

Open Confirma on your phone and add it to your home screen.

Confirma is delivered as a Progressive Web App, which means it works on Android and iPhone without going through an app store. After installation, it can keep working offline.

Android / Chrome

Open the app and install when prompted.

Visit your live Confirma URL in Chrome on Android. Chrome should show an install prompt automatically, or the user can use the browser menu to install the app.

Open Android app link
  • Open the production URL
  • Tap the install prompt if it appears
  • If needed, use the Chrome menu and choose install
  • Launch Confirma from the home screen after installation
iPhone / Safari

Use “Add to Home Screen.”

Open the production URL in Safari. Confirma can show guidance telling users to tap the Share button and then choose “Add to Home Screen.”

Open iPhone app link
  • Open the production URL in Safari
  • Tap the Share button
  • Select “Add to Home Screen”
  • Launch Confirma like a regular app from the home screen
