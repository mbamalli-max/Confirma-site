Institutional relevance

Why this is different from spreadsheets and self-reported summaries.

Confirmation-gated authority

Records are not formed by passive autosave or later admin intervention. The user must explicitly confirm before the transaction becomes permanent.

Append-only structure

There is no edit path and no delete path. A new entry can be added, but an existing confirmed entry cannot be rewritten or removed.

Verifiable export surface

Free exports include a ledger root hash and integrity code. Phase 2 expands this into server-attested verified reports and live verification flows.

Who it serves

Built for underserved but economically active businesses.

Primary focus begins with Nigeria's informal economy: market traders, food vendors, artisans, transport operators, small shops, and service providers. These businesses often have real cash flow but insufficient formal records for conventional credit review.

Confirma gives institutions a path to evaluate more than absence. It creates a structured history where none existed before.

Underwriting value

Context lenders can use.

  • Observe durability through repeated use over 30, 90, and 180-day periods
  • Inspect transaction continuity rather than relying only on one-time declarations
  • Use trust tiers and history depth as additional underwriting context
  • Prepare for future verified reports, server sync, and fork detection in Phase 2
Roadmap

From local trust formation to verified reporting.

Phase 1 — Complete

Offline ledger, voice and text input, dashboard, export, onboarding, multilingual support, and country-aware configuration.

Phase 2 — Next

Device-bound Ed25519 signatures, server sync, fork detection, paid verified reports, and a live verification portal.

Phase 3

Bank API integration, loan-readiness notifications, agent verification workflows, and iOS native support.

Strategic outcome

A new trust layer for informal and interaction-constrained economic participation, starting in Nigeria and designed for broader deployment.

"No server edits. No admin overrides. No hidden paths. Only you can confirm." For institutions, that user-controlled formation model is what makes the record more meaningful.
