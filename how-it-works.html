Simple explanation of the cryptographic chain

Each confirmed transaction becomes part of a protected sequence.

Confirma is built so records are formed carefully, then kept permanently. The goal is not to overwhelm users with technical terms. The goal is to make trust understandable.

Core principle

Confirma does not treat typing as authority. Confirmation is the moment a record is formed.

Step by step

What happens when a user records a transaction?

1. The user enters a transaction

A business owner speaks or types something like “Sold rice for 15,000.”

2. Confirma interprets it

The app parses the transaction and reads back a plain-language summary before anything is written permanently.

3. The user confirms

This is the decisive step. Without confirmation, there is no permanent entry.

4. A new record is created

The entry gets a unique ID, a confirmation timestamp, and a cryptographic link to the previous entry in the ledger.

Append-only logic

Why records cannot be silently changed.

After confirmation, the transaction is not editable. It is not deletable. If another event happens later, that event becomes a new record instead of rewriting the old one.

This matters because institutions need to know that historical records have structure and continuity, not just convenience.

Hash chaining

How the chain works in simple terms.

Every record contains a fingerprint of the previous record. That fingerprint is created using SHA-256. If someone tried to tamper with an older entry, the chain after it would no longer match properly.

That is what makes the ledger tamper-resistant and more useful for verification.

Verification surface

What can be checked today?

🧮

Ledger root hash

The export includes the final hash in the visible chain for the exported period.

🧷

Integrity code

The export also includes an integrity code derived from the included transaction hashes, supporting offline checking.

📤

Exportable history

Users can generate a free export of the last 30 days of confirmed transactions to share or preserve.
