# Theme Factory

## Overview

Offer a small set of distinct visual directions, let the user choose, then apply the selected theme consistently. If the preset options are not a fit, generate a custom theme with the same level of specificity.

## Workflow

1. Identify the artifact being styled and any audience or mood cues from the request.
2. If a bundled showcase exists, present it for selection without editing it.
3. If no showcase exists, summarize the available preset themes in plain language and ask the user to choose one.
4. Wait for an explicit selection before applying a preset theme.
5. If none fit, generate one custom theme, show it for review, and apply it only after confirmation.

## Preset Themes

- Ocean Depths: calm, maritime, professional blues
- Sunset Boulevard: warm, saturated sunset tones
- Forest Canopy: grounded greens and earth tones
- Modern Minimalist: restrained grayscale and clean typography
- Golden Hour: rich amber and autumn warmth
- Arctic Frost: cool, crisp, high-clarity tones
- Desert Rose: dusty, soft, sophisticated warmth
- Tech Innovation: bold, modern, high-contrast technology feel
- Botanical Garden: fresh organic greens with lively accents
- Midnight Galaxy: dramatic deep tones with cosmic contrast

## Applying A Theme

- Translate the chosen theme into a concrete palette, heading font, body font, and accent usage plan.
- Apply the theme across the full artifact, not just the cover or first section.
- Maintain readability and contrast even when the theme is expressive.
- Keep type choices limited and intentional; one heading family and one body family are usually enough.
- Preserve the artifact's content and hierarchy while changing its visual identity.

## Generating A Custom Theme

When creating a custom theme:

1. Name it with a short evocative phrase.
2. Define a palette with clear roles for background, text, primary accent, and secondary accent.
3. Choose heading and body fonts that match the requested mood.
4. Summarize the intended visual character in 2-4 sentences.
5. Present the theme for approval before applying it.

## Resource Expectations

- If the skill bundle includes `theme-showcase.pdf`, use it as the first-choice selection surface.
- If the skill bundle includes theme definition files, read the selected one before styling.
- If those bundled assets are absent, recreate the same workflow conversationally instead of blocking.
