# Canvas Workflow

Use this guide when executing `$canvas-design` on a real request.

## Philosophy File

Create the philosophy first and save it as markdown.

### Structure

1. Name the movement in 1-2 words.
2. Write 4-6 paragraphs that describe the visual worldview.
3. Focus on how the philosophy appears through space, form, color, scale, rhythm, composition, hierarchy, and restraint.

### Emphasis

- Keep the philosophy visual-first, not narrative-first.
- Use minimal text as a repeated design principle.
- Leave room for interpretation instead of prescribing a rigid layout.
- Stress meticulous craftsmanship and master-level execution throughout the writing.
- Avoid repeating the same point unless new nuance is added.

## Subtle Reference

Before designing the canvas, identify the quiet conceptual thread inside the request.

- Treat the topic as a subtle embedded reference, not a literal slogan.
- Make the work legible and compelling even for viewers who miss the reference.
- Let people who know the subject feel it intuitively through the composition.

## Canvas Creation

Create one highly polished page by default.

### Composition

- Build meaning through form, space, rhythm, repeated elements, or layered systems.
- Favor shapes, graphics, texture, image treatment, and visual hierarchy over paragraphs of text.
- Let the piece feel like an artifact from an imagined discipline: systematic, observed, and intentional.
- Use a limited palette unless the concept truly benefits from broader chromatic range.

### Typography

- Keep text sparse and integrated into the composition.
- Let context control scale: some prompts may want whisper-quiet labels, others may support bolder poster typography.
- Prefer thin or restrained type unless the concept calls for forceful typographic gestures.
- Search `./canvas-fonts` first if that directory exists and the design would benefit from its fonts.
- Keep all text inside the canvas with safe margins and no overlaps.

### Quality Bar

- Make the piece feel museum-quality or magazine-quality, never generic AI filler.
- Refine spacing, alignment, edge handling, and contrast until the composition feels deliberate.
- Avoid adding extra graphics just to make the page busier.

## Final Refinement Pass

Take a second pass before finishing.

1. Remove or simplify anything that weakens the composition.
2. Tighten spacing, scale relationships, and color balance.
3. Improve crispness and cohesion before introducing anything new.
4. Verify that every element has breathing room and nothing collides or falls off the page.

## Multi-Page Requests

When the user asks for additional pages:

- Keep the same philosophy and conceptual thread.
- Make each new page a distinct variation, not a duplicate.
- Treat the set like a coherent series with evolving visual ideas.
